프로젝트 관리

프로젝트 패널 폴더 구조

프로젝트 패널을 체계적으로 정리해두면 소재를 찾는 시간을 줄이고 협업도 쉬워집니다. 처음부터 폴더를 만들어두는 습관이 중요합니다.

📁

Subcomp

메인 컴포지션을 제외한 나머지 컴포지션을 정리하는 폴더. 중간 과정의 서브 컴포지션들이 여기에 들어간다.

예) 캐릭터 움직임 컴프, 배경 컴프, 이펙트 컴프 등
📂

Source

외부에서 가져온 편집 재료를 정리하는 폴더. 영상, 이미지, 음악, 효과음 등 외부 소스 파일이 여기에 들어간다.

예) 촬영본 mp4, 배경음악 mp3, 로고 png 등
🟫

Solid

AE에서 생성한 단색 레이어(Solid)를 따로 모아두는 폴더. 배경이나 마스크용 솔리드가 여기에 정리된다.

AE가 자동으로 Solids 폴더를 만들기도 함
정리 원칙 메인 컴포지션(최종 출력용)은 패널 최상단에 단독으로 두고, 나머지는 모두 폴더 안으로 정리한다. 작업 초반에 폴더를 먼저 만들어두는 것이 핵심.
FPS 관리

영상 소스 FPS — 반드시 확인해야 하는 규칙

FPS가 맞지 않으면 재생 속도가 틀어지거나 편집점이 어긋납니다. 소스를 가져올 때 FPS 확인은 선택이 아닌 필수입니다.

RULE 1
영상 소스를 가져올 때 꼭 FPS를 확인한다

소스 파일을 임포트하면 해당 영상의 FPS를 프로젝트 패널에서 확인할 수 있다. 메인 컴포지션과 맞는지 즉시 체크.

RULE 2
메인 컴포지션 FPS = 소스 FPS — 반드시 일치!

최종 렌더링하는 메인 컴포지션의 FPS와 영상 소스의 FPS가 다르면 절대 안 된다. 다를 경우 속도가 왜곡되어 재생된다.

FPS 불일치 예시
❌ 잘못된 경우
메인 컴프: 30fps 소스 영상: 24fps

재생 속도 왜곡, 편집점 어긋남

✓ 올바른 경우
메인 컴프: 30fps 소스 영상: 30fps

정확한 속도와 편집점 유지

편집 단축키

편집 실습 핵심 단축키

편집 속도를 결정하는 단축키입니다. 마우스 없이 키보드만으로 타임라인을 제어할 수 있어야 합니다.

재생 제어

영상 재생 / 정지

타임라인에서 현재 위치부터 재생. RAM 프리뷰와 다름.

Spaceor0(숫자패드)

레이어 편집

레이어 자르기 (Split Layer)

인디케이터(재생헤드) 위치에서 레이어를 두 개로 분리. 영상 편집의 가장 기본 동작.

Ctrl+Shift+D
레이어 앞부분을 인디케이터로 이동

레이어를 자른 뒤 위치를 재배치할 때. 레이어의 In 포인트(시작)가 인디케이터 위치로 스냅.

[
레이어 뒷부분을 인디케이터로 이동

레이어의 Out 포인트(끝)가 인디케이터 위치로 스냅. 레이어 끝을 맞출 때 사용.

]

인디케이터 이동

1프레임씩 이동

정밀한 편집점 확인 시 사용. 컷 포인트를 한 장씩 확인할 때 필수.

PgUp/PgDn
10프레임씩 이동

타임라인에서 빠르게 앞뒤로 이동할 때. 긴 타임라인 탐색에 유용.

Shift+PgUp/PgDn
인디케이터를 편집점에 스냅

Shift를 누른 채 인디케이터를 드래그하면 레이어의 편집점(In/Out)에 자동으로 달라붙는다.

Shift+드래그
편집점 팁

편집점 정밀 작업 — 뒷부분 1프레임 규칙

편집이 정확하게 딱 붙지 않는 원인 중 하나입니다. 편집점 앞부분과 뒷부분의 스냅 동작이 다릅니다.

앞부분 편집점
✓

Shift + 드래그로 인디케이터를 레이어 시작점에 스냅하면 정확하게 맞는다.

그대로 자르면 OK
뒷부분 편집점
⚠️

그냥 자르면 1프레임 남음!
