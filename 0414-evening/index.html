Mix Material

믹스 메터리얼 — 여러 재질을 레이어처럼 쌓기

Mix Material은 두 개의 재질을 블렌딩해 하나의 새로운 재질을 만드는 기능입니다. 이를 반복적으로 중첩하면 복잡한 재질 표현이 가능합니다.

🔩
이번 작업 목표

일반 강철 → 녹슨 강철 → 페인트칠 → 다시 녹슨 강철이 드러나는 레이어드 녹슨 금속 재질 완성

🎨
Mix Material 사용 원리

A + B = Mix → 이 Mix가 다시 C와 섞여 새로운 Mix. 마스크(흑백 이미지 또는 노이즈)로 두 재질의 경계를 제어.

재질 준비

베이스 재질 3종 준비

믹스에 사용할 원본 재질을 먼저 준비합니다. 폴리헤븐(Poly Haven)과 쉐어텍스처(Share Textures)에서 무료로 다운로드 가능합니다.

01

녹슨 강철

Metallic

표면이 부식된 러스트 텍스처. 최종 결과물에서 틈새와 가장자리에 드러나는 핵심 레이어.

📦 polyhaven — rusty_metal_03_2k
02

일반 강철

Metallic

베이스가 되는 깔끔한 금속 표면. 믹스의 시작점으로 사용.

📦 sharetextures — metal_9-dOu-2K
03

페인트 강철

Glossy

파란 페인트가 칠해진 금속판. 중간 레이어로 사용되며 벗겨진 느낌 표현에 핵심.

📦 polyhaven — blue_metal_plate_2k
텍스처 다운로드 팁 폴리헤븐은 Diffuse·Roughness·Normal·Displacement 맵을 한 번에 제공합니다. 2K 해상도로 다운로드하면 용량과 품질의 균형이 좋습니다.

잔디·꽃 기법 ①

흑백 이미지 + Opacity — 플레인 기법

실제 3D 모델링 없이 플레인(면) 하나에 흑백 이미지와 Opacity를 조합해 풀이나 꽃처럼 보이게 만드는 가벼운 기법입니다.

1
풀·꽃 흑백 이미지 준비

배경이 없는(투명 PNG 또는 흑백) 식물 이미지를 구한다. 흰 부분 = 보임, 검은 부분 = 투명.

2
플레인 오브젝트에 Octane 재질 적용

단순한 사각 면(Plane)을 만들고, 이미지를 Opacity 채널에 연결해 불필요한 영역을 투명하게 처리.

3
복제(복붙)해서 배치

만들어진 플레인을 클로너 또는 수동으로 여러 개 복제해 바닥에 배치하면 풀밭·꽃밭처럼 보임.

⚠️ 사용 제한
❌ 이 방법을 쓰면 안 되는 경우

카메라가 가까이 확대(줌인)하는 씬. 플레인이 납작하게 보여 티가 남.

✓ 이 방법이 통하는 경우

멀리서 바라보는 씬. 멀리서 봤을 때는 충분히 자연스럽고 렌더 속도도 빠름.

Best Tip 이미지 활용을 잘하면 모델링 없이도 충분히 현실적인 장면을 만들 수 있습니다. 폴리곤 수가 적어 렌더가 빠르다는 것도 큰 장점입니다.
잔디·꽃 기법 ②

Forester 플러그인 — 고품질 자연 오브젝트

Forester는 C4D용 식물·자연 요소 생성 플러그인입니다. 고품질 풀, 꽃, 나무 등을 쉽게 배치할 수 있지만 Octane과 함께 쓸 때 몇 가지 처리가 필요합니다.

⚠️ 01
Octane 재질로 변환 필수

Forester로 생성된 오브젝트는 기본적으로 C4D 기본 재질로 나옵니다. Octane으로 렌더하려면 재질을 Octane용으로 교체해야 합니다.

🔧 02
면 정리 (폴리곤 수 감소)

기본 상태에서는 면의 개수가 너무 많아 렌더 속도가 느립니다. 면 수를 줄여 최적화 필요.

💡 03
이미지 경로 복붙으로 텍스처 연결

C4D 기본 재질에 들어있는 이미지 파일 경로를 확인한 뒤, 해당 경로를 Octane 재질의 Image Texture 노드에 붙여넣어 수동으로 연결합니다.

다소 반복 작업(노가다)이 필요하지만, 모델링을 직접 하지 않아도 되므로 이 정도는 감수할 가치가 있다!
