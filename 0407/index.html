상단 모델링

립스틱 상단 디테일 — 구멍 뚫기 & 루즈 탑 만들기

01
실린더 뚜껑에 두께 + 구멍 만들기
합쳐진 상단 오브젝트에서 실린더 뚜껑 면을 선택한 뒤, Extrude Inner → Extrude 순서로 두께감을 주면서 구멍을 뚫어줍니다.
IExtrude Inner — 면 안으로 줄이며 테두리 생성
DExtrude — 면을 밀어 넣어 구멍 형태 완성
02
Split으로 면 분리 복사
구멍을 뚫은 면이 선택된 상태 그대로 UP(Split) 기능을 사용해 해당 면을 별도 오브젝트로 분리·복사합니다. 이 복사된 면이 루즈(립스틱 내용물)의 기반이 됩니다.
UPSplit — 선택 면을 분리해 복사
03
Extrude 3회 + Scale 줄이기 — 둥근 탑 형태

Subdivision Surface

Subdivision Surface 적용 & 각잡기

완성된 립스틱 오브젝트를 Subdivision Surface에 넣으면 표면이 부드럽게 말립니다. 하지만 너무 둥글어지면 형태가 무너지므로, 유지하고 싶은 엣지 주변에 선을 추가해 각을 잡아야 합니다.

각잡기 방법 — 원하는 라인 선택 후 주변에 선 추가
Subdivision Surface는 엣지 사이의 간격이 넓을수록 더 둥글게 처리합니다. 각지게 유지하고 싶은 엣지 바로 옆에 선을 하나 더 추가하면 Subdivision이 해당 부분을 덜 둥글게 처리합니다.
MS
Bevel
주로 사용
선택한 엣지를 양쪽으로 벌려 두 개의 선으로 만드는 기능입니다. 각도를 유지하면서 자연스럽게 선을 추가합니다.
⚠ Bevel 모드는 반드시 Solid 모드로 진행
MK
Line Cut
원하는 위치에 자유롭게 선을 잘라 추가합니다. 정확한 위치에 선을 하나씩 넣을 때 사용합니다.
ML
Loop Path Cut
오브젝트를 한 바퀴 도는 루프 형태로 선을 추가합니다. 전체적으로 균일하게 선을 넣을 때 효율적입니다.
