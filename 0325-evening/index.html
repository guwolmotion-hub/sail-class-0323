기초 개념

UV Mapping이란?

3D 오브젝트에 이미지 텍스처를 입히려면 단순히 이미지를 갖다 붙이는 것이 아닙니다. 입체 형태를 2D로 "펼쳐서" 텍스처를 배치하는 과정이 필요한데, 이것이 바로 UV Mapping입니다.

UV가 잘 펼쳐져 있어야 텍스처가 왜곡 없이 오브젝트에 자연스럽게 입혀집니다. UV가 겹치거나 뭉개져 있으면 텍스처가 이상하게 늘어나거나 끊기는 현상이 발생합니다.

조명 기법

Gobo Light — 이미지로 그림자를 만드는 조명 기법

Gobo Light는 흑백 텍스처 이미지를 조명의 Distribution(분포) 채널에 연결해 흰색 부분에서만 빛이 나오게 만드는 기법입니다. 실제 사진 스튜디오에서 조명 앞에 패턴 판(Gobo)을 세워 그림자를 연출하는 것과 같은 원리입니다.

왜 Gobo Light를 사용하나요?

흑백 텍스처의 역할

Distribution에 연결된 흑백 이미지에서 흰색 부분 = 빛이 통과 (밝음), 검정 부분 = 빛이 차단 (그림자)으로 작동합니다. 창문 블라인드, 나뭇잎 그림자, 격자 패턴 등 다양한 흑백 이미지를 활용해 분위기 있는 조명 연출이 가능합니다.

Step by Step

Gobo Light 적용 전체 순서

Gobo Light 설정은 단계별로 순서대로 진행해야 합니다. 특히 Projection 변경과 Border Mode 설정은 빠뜨리기 쉬운 핵심 단계입니다.

01
흑백 텍스처를 Image Texture 노드에 불러오기
원하는 패턴의 흑백 이미지(블라인드, 나뭇잎, 격자 등)를 Image Texture 노드에 불러옵니다. 흑백 이미지일수록 빛과 그림자의 경계가 명확하게 표현됩니다.
02
Image Texture 노드 → Distribution에 연결
Image Texture 노드의 출력을 조명 오브젝트의 Distribution 채널에 연결합니다. 이것만으로 흑백 이미지의 패턴대로 빛이 나오기 시작합니다.
이 시점에서는 이미지가 올바르게 보이지 않는 것이 정상입니다 →
!
왜 이미지가 이상하게 보이나요?
조명 오브젝트는 일반 오브젝트와 달리 메쉬(폴리곤)가 없습니다. 메쉬가 없으면 UV 매핑을 할 수 없기 때문에 이미지가 제대로 투영되지 않습니다. 이를 해결하려면 프로젝션 방식을 직접 변경해야 합니다.
03
Projection을 Mesh UV → Perspective로 변경
Image Texture 노드의 Projection(투영) 설정을 기본값인 Mesh UV에서 Perspective로 변경합니다. 이렇게 하면 조명이 이미지를 원근감 있게 앞에서 투영하는 방식으로 작동합니다.
04
UV Transform으로 이미지 크기 조절
UV Transform 노드를 연결하여 이미지의 크기(Scale), 위치(Offset), 회전(Rotation)을 조절합니다. 빛이 닿는 영역에서 이미지 패턴이 원하는 크기로 나타나도록 설정합니다.
05
타일링 발생 시 Border Mode 설정
이미지 범위를 벗어난 영역이 반복(타일링)되는 경우, Image Texture 노드의 Border Mode(테두리 색상)를 적절히 변경합니다. 아래 표를 참고하여 상황에 맞게 선택하세요.
↓ Border Mode 상세 설명 참고
06
조명의 위치 · 크기 · 각도 최종 조정
모든 노드 설정이 끝나면 조명 오브젝트 자체의 위치, 크기, 각도를 조정하여 그림자가 원하는 위치와 크기로 생성되도록 최종 마무리합니다.

Projection 설정 — Mesh UV vs Perspective

조명 오브젝트는 메쉬가 없어 UV 매핑이 불가능합니다. 이미지 텍스처 노드의 Projection 방식을 바꾸어 이 문제를 해결합니다.

타일링 해결

Border Mode — 이미지 범위 밖을 어떻게 처리할 것인가

이미지가 조명 영역보다 작을 경우, 이미지 범위를 벗어난 부분에서 타일링(반복)이 일어날 수 있습니다. 이때 Image Texture 노드의 Border Mode(UV 테두리 색상) 설정으로 이를 해결합니다.

배경 상황
Border Mode 설정
이유
배경이 검정색인 경우
Black Color
이미지 범위 밖을 검정으로 채워 배경과 자연스럽게 이어집니다.
배경이 흰색인 경우
White Color
이미지 범위 밖을 흰색으로 채워 배경과 자연스럽게 이어집니다.
이미지가 컬러 이미지인 경우
Clamp Value
이미지 테두리의 마지막 픽셀 색상으로 채워 자연스럽게 확장됩니다.
💡
타일링이 발생하는 원인

이미지 텍스처 노드는 기본적으로 UV 좌표 0~1 범위에 이미지를 표시하고, 범위를 벗어나면 이미지를 반복(타일)합니다. Perspective 투영에서는 조명 각도에 따라 이미지 범위 밖까지 투영될 수 있어 Border Mode 설정이 필요합니다.

마무리

최종 마무리 — 조명 위치 · 크기 · 각도 조정

노드 설정이 완료되었다면, 이제 조명 오브젝트 자체를 움직여 그림자의 형태를 완성합니다. 조명의 세 가지 물리적 속성이 Gobo 그림자에 어떤 영향을 미치는지 이해하고 세밀하게 조절해보세요.

📍
위치 (Position)
조명을 앞뒤·좌우·상하로 이동하면 그림자가 투영되는 위치가 바뀝니다. 조명이 피사체에 가까울수록 그림자가 커지고, 멀수록 작아집니다.
⬛
크기 (Size)
조명 오브젝트의 크기를 조절하면 그림자의 전체 스케일이 바뀌고, 소프트/하드 그림자 정도도 함께 변합니다.
📐
각도 (Angle)
조명을 회전하면 그림자 패턴의 기울기와 방향이 달라집니다. 빛이 비스듬히 들어올수록 그림자가 늘어나며 더 극적인 연출이 가능합니다.
