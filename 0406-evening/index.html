몬스터볼 — 모델링부터 텍스처까지

모델링

몬스터볼 모델링 — 7단계

Sphere 오브젝트 하나에서 시작해 필요 없는 면을 제거하고, Extrude로 두께를 만들고, Symmetry로 아래쪽을 자동 완성하는 효율적인 워크플로우입니다.

01
Sphere 생성 — 세그먼트 & 방향 설정
Sphere를 꺼내고 세그먼트를 높게 설정해 표면을 부드럽게 만듭니다. 그런 다음 로테이션 90도 회전 — 뚜껑 경계선이 정면을 바라보도록 맞추기 위해서입니다.
Segments 32 ~ 64 Rotation 90°
기본 Sphere는 경계선이 옆을 향해 절반 선택이 어렵습니다. 90도 회전하면 경계선이 정면이 되어 정확하게 절반을 선택·삭제할 수 있습니다.
02
절반 제거 — 뚜껑 반구 만들기
Make Editable 후 폴리곤 모드에서 아래쪽 절반을 선택해 삭제합니다. 이어서 뚜껑의 둥근 윗부분 일부도 추가 삭제합니다. 이 공간에 나중에 버튼(실린더)이 들어갑니다.
03
Extrude — 두께감 만들기
반구 오브젝트에 Extrude를 적용해 실제 플라스틱 케이스 같은 두께를 만듭니다.
Create Caps ✓ 체크 필수
Create Caps를 체크해야 Extrude 후 단면이 막힌 상태로 두께가 생깁니다. 체크 안 하면 속이 빈 껍데기가 됩니다.
04
Symmetry — 하단 자동 복사
뚜껑을 Symmetry로 감싸면 XZ 평면을 기준으로 아래쪽으로 자동 복사되어 몬스터볼 하단 본체가 완성됩니다. 뚜껑을 수정하면 하단도 실시간으로 함께 바뀝니다.
Mirror Plane: XZ
05
경계 Sphere 추가
Sphere를 하나 더 생성해 뚜껑과 하단이 만나는 가운데 경계에 배치합니다. 크기를 뚜껑과 정확히 맞춰 몬스터볼 중앙 경계선 디테일을 만듭니다.
06
실린더로 버튼 제작
실린더로 중앙 버튼을 만듭니다. 앞서 지운 둥근 부분보다 약간 더 작게 크기 조절 후 윗면에 Extrude Inner → Extrude 순서로 버튼 돌출 형태를 완성합니다.
IExtrude Inner — 면을 안으로 줄이며 테두리 생성
DExtrude — 면을 뽑아 버튼 돌출 완성
07
전체 합치기 — Connect Objects
뚜껑, Symmetry 본체, 경계 Sphere, 버튼 실린더를 모두 선택해 하나의 오브젝트로 합칩니다. 이후 텍스처 작업이 훨씬 편해집니다.

Dirt / Grunge

Dirt & Grunge — "너무 깨끗하면 CG처럼 보인다"

완벽하게 깨끗한 표면은 오히려 가짜처럼 보입니다. Dirt와 Grunge 레이어를 얇게 얹으면 사용감, 오염, 마모가 생겨 훨씬 현실적인 질감이 됩니다.

Dirt
먼지 · 오염 · 변색
오브젝트의 구석, 홈, 움푹 들어간 부분에 자동으로 집중되는 오염 효과입니다. 빛이 닿기 어려운 영역에 자연스럽게 어두운 오염이 생겨 입체감을 강화합니다.
틈새 먼지구석 그림자오래된 변색
Grunge
긁힘 · 마모 · 얼룩
노이즈 패턴 기반의 불규칙한 마모·긁힘을 표면에 입힙니다. 무작위로 분포되어 인위적으로 만들기 어려운 자연스러운 질감을 빠르게 표현합니다.
페인트 벗겨짐긁힌 금속닳은 모서리
💡
강도는 낮게 — 불투명도 10~30%가 적당

처음엔 효과를 강하게 주고 싶어지지만, 자연스러운 오염과 마모는 생각보다 매우 미묘합니다. 불투명도를 낮게 유지하면서 렌더링으로 직접 확인하며 조금씩 올리는 방식으로 작업하세요.
