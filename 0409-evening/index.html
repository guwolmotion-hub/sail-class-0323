멀티패스 합성

C4D 렌더링 후 AE 후반작업

C4D에서 출력된 이미지는 색감이 쨍하지 않습니다. 멀티패스로 분리한 레이어들을 AE에서 블렌딩 모드로 합성해 색감을 살립니다.

2D 영상 표준 29.97 fps 드랍 프레임(Drop Frame)
논드랍 30 fps Non-Drop Frame
C4D 색감 특성 C4D 기본 렌더 이미지는 색감이 쨍하게 잡히지 않습니다. 멀티패스를 활용한 AE 합성과 색보정 작업이 필수입니다.

멀티패스 합성 구조

최상위 언멀트(Unmult) 적용 레이어 검은 영역을 제거한 후 블렌딩 모드 적용
Reflection Direct / InDirect 블렌딩 모드: Screen — 반사감 강화
Diffuse Direct / InDirect 블렌딩 모드: Soft Light / Screen / Opacity
베이스 메인 렌더 이미지 C4D 최종 출력 이미지
블렌딩 모드

멀티패스 합성용 블렌딩 모드 가이드

같은 Diffuse 패스라도 어떤 블렌딩 모드를 쓰느냐에 따라 결과가 완전히 달라집니다. 상황에 맞게 선택하세요.

Soft Light

이미지 전체 색감의 대비(Contrast)를 강하게 주고 싶을 때

  • 어두운 부분은 더 어둡게, 밝은 부분은 더 밝게
  • 드라마틱한 느낌 연출에 적합
Diffuse에 주로 사용
Screen

색상 표현이 더 밝아졌으면 할 때

  • 이미지 전체를 밝혀주는 효과
  • 여러 번 덮으면 더 강해짐 (반사 강조에 활용)
Diffuse & Reflection 모두 사용
Opacity (불투명도)

원본 색감을 그대로 유지하면서 좀 더 쨍하게

  • 색 변환 없이 채도·선명도만 높이는 느낌
  • 수치를 낮춰 미묘하게 조정할 때 유용
Diffuse에 사용
언멀트(Unmult) 플러그인

패스 이미지의 검은 영역을 투명하게 제거한 뒤 블렌딩 모드를 동일하게 적용

  • 검은 배경이 합성에 영향을 주는 문제 해결
  • 더 깔끔한 합성 결과물
플러그인 필요
Reflection 패스 활용법 Diffuse 보정을 하고 나면 반사가 많이 죽습니다. 이때 Reflection Direct/InDirect 패스를 Screen 모드로 올려 반사를 되살리거나, 더 강한 반사 이미지를 원할 때 여러 번 덮어 사용합니다.
색보정

AE 색보정 효과 총정리

3D 결과물 보정 시 자주 쓰이는 AE 이펙트입니다. 정밀도 순서로 이해하면 용도가 명확해집니다.

전체 명도를 올리거나 내림. 흰색·검정을 이미지에 덮는 느낌 — 톤 유지 X

Curves ★
명도 보정 중 가장 세밀하게 조절 가능. RGB 개별 채널 제어. 실무 표준
최고

블러 & 선명도

빛 보정 & 선명도 조정 — Blur 이펙트

DoF 연출, 모션 블러, 글로우 효과 등 다양한 목적으로 사용되는 블러 이펙트입니다.

Fast Box Blur

가장 빠른 블러. 렌더 속도가 중요할 때 사용. 품질은 낮지만 부드러운 효과에 적합

Gaussian Blur

가장 일반적인 블러. 자연스러운 흐림 효과. 배경 소프트 처리에 많이 사용

Camera Lens Blur

실제 카메라 렌즈의 DoF(피사계심도)를 시뮬레이션. 보케(Bokeh) 효과 포함. 가장 현실적

Radial Fast Blur

중심점에서 방사형으로 퍼지는 블러. 줌 인/아웃 모션, 속도감 표현에 적합

Directional Blur

특정 방향으로만 흐림 처리. 수평·수직 모션 블러, 속도감 연출에 활용

노이즈

노이즈 추가 & 제거

깨끗한 3D 렌더는 오히려 부자연스러울 수 있습니다. 적절한 노이즈를 추가해 사실감과 질감을 높일 수 있습니다.

Noise HLS ★

색상(Hue)·명도(Lightness)·채도(Saturation) 채널을 이용해 노이즈를 만드는 이펙트입니다.

추천 사용법 명도(L)나 채도(S) 채널만 활성화해서 사용하면 색 변화 없이 자연스러운 필름 그레인 느낌을 만들 수 있습니다.

Add Grain

필름 그레인 노이즈를 추가. 다양한 필름 종류 시뮬레이션 파라미터 제공. 아날로그 감성 표현에 적합

Fractal Noise

프랙탈 기반의 복잡한 노이즈 패턴 생성. 연기·안개·텍스처 배경 등 다양하게 활용 가능

AE 필수 플러그인 목록

🎨

Magic Bullet Looks

영화·광고 수준의 색보정 플러그인. 프리셋 기반의 시네마틱 룩 적용이 쉬움

유료
🌟

Universe — Finisher / Glow

다양한 전환·글로우·스타일 이펙트 모음. Finisher는 원클릭 마무리 효과로 유명

유료
💡

Deep Glow

유료
우선순위 당장 설치해야 할 필수 무료 플러그인: 언멀트 + Fx Console. 이 두 가지만 있어도 AE 3D 후반작업 효율이 크게 올라갑니다.
