Xpresso 개요

Xpresso — C4D의 노드 기반 자동화 시스템

After Effects의 익스프레션(Expression)과 같은 역할을 하지만, 코드를 직접 입력하는 방식이 아닌 노드를 연결하는 방식으로 작업합니다. 오브젝트 간의 관계를 설정해 하나의 값이 바뀌면 연결된 다른 값들이 자동으로 따라 움직이게 만드는 기능입니다.

유저데이터

User Data — 컨트롤러 슬라이더 만들기

Xpresso에서는 보통 Null 오브젝트에 유저데이터를 추가해 컨트롤러로 사용합니다. 유저데이터의 Unit(단위) 설정이 Xpresso 노드 연결 시 Range Mapper의 입력 범위를 결정합니다.

Percent
0 ~ 100%
가장 직관적인 컨트롤러. 0%에서 100%까지 슬라이더로 제어. Xpresso 컨트롤러로 가장 많이 사용.
Degree
각도값 (°)
회전 관련 수치를 다룰 때 사용. Rotation에 직접 연결할 때 편리.
Length Unit
거리값 (cm)
Position이나 Scale 거리를 cm 단위로 직접 입력할 때 사용.
Real
실수값
단위 없는 순수 숫자. 비율 계산이나 Math 노드 입력에 활용.
Xpresso 노드

노드의 기본 구조 — 빨간 버튼 & 파란 버튼

Xpresso의 모든 노드는 빨간 포트(출력)와 파란 포트(입력)으로 구성됩니다. 빨간 포트에서 선을 뽑아 파란 포트에 연결하는 것이 Xpresso 작업의 전부입니다.

빨간 버튼 — 출력 (Output)
수치를 내보내는 포트입니다. 동시에 여러 갈래로 내보낼 수 있습니다.
1 → 여러 개 연결 가능 ✓
→
파란 버튼 — 입력 (Input)
수치를 받아들이는 포트입니다. 하나의 파란 포트에는 하나의 연결만 가능합니다.
하나만 연결 가능 — 두 개 연결 시 나중 것이 덮어씀
핵심 노드 2가지
★★★ 필수
Range Mapper
서로 다른 단위와 범위를 가진 수치들을 하나로 연결할 때 사용합니다. 예를 들어 0~100%(컨트롤러)를 0~500cm(이동 거리)로 변환하거나, 0~100%를 0~360°(회전)로 변환하는 식으로 범위를 매핑합니다.
입력: 0 ~ 100% → 출력: 0 ~ 500cm
입력: 0 ~ 100% → 출력: 0 ~ 360°
수학 연산
Math
수학 공식이 필요할 때 사용합니다. 두 수치를 더하거나, 특정 값을 곱해 스케일을 조정하는 등 연산이 필요한 상황에 노드 사이에 끼워 사용합니다.
Add 더하기 Subtract 빼기 Multiply 곱하기 Divide 나누기
실습 예시

컨트롤러 하나로 Scale · Position · Rotation 동시 제어

Null 레이어에 퍼센트 유저데이터를 만들어 컨트롤러로 사용하고, 이 값에 큐브의 스케일·포지션·로테이션을 한 번에 연동하는 기본 Xpresso 구성입니다.

세팅 조건
①
컨트롤러 — Null 레이어에 유저데이터 추가. 단위: Percent (0~100%)
②
Scale 연동 — 컨트롤러 0~100% → 큐브 Scale 0~1
Range Mapper: 입력 0~100 / 출력 0~1
③
Position 연동 — 컨트롤러 0~100% → 큐브 Position X 0~500cm
Range Mapper: 입력 0~100 / 출력 0~500
④
Rotation 연동 — 컨트롤러 0~100% → 큐브 Rotation H/P/B 각각 0~360°
Range Mapper 3개: 각 축마다 입력 0~100 / 출력 0~360

퀴즈

실습 퀴즈 — 큐브 A · B 반대 방향 연동

조건 설정
컨트롤러 Null 레이어 생성 → 유저데이터: 퍼센트 (0~100%)
큐브 오브젝트 2개를 컨트롤러 안에 배치 → 큐브 A, 큐브 B
큐브 A
Position X 0 → -1000cm (감소)
Rotation B 0 → +250°
vs
큐브 B
Position X 0 → +1000cm (증가)
Rotation B 0 → -250°
💡
A와 B가 반대 방향으로 움직이도록 Range Mapper의 출력 범위를 반전시키면 됩니다. A: 출력 0~-1000 / B: 출력 0~1000 — Rotation도 마찬가지로 부호만 반전.
립스틱 모델링

립스틱 케이스 — 큐브에서 상/하단 오브젝트 분리

큐브 오브젝트를 기반으로 립스틱 케이스의 기본 형태를 만들고, 상단(캡)과 하단(본체)을 분리하는 과정입니다. 모델링 단축키들을 정확히 익혀두는 것이 핵심입니다.

01
큐브 오브젝트 기본 세팅
큐브 오브젝트를 생성하고 크기와 세그먼트를 조정합니다.
크기 Y300cm
세그먼트 X2
세그먼트 Y5
세그먼트 Z2
02
Bulge 디포머 적용
큐브에 불지(Bulge) 디포머를 추가해 립스틱 특유의 볼록한 형태를 만듭니다.
Fit to Parent적용
Strength25%
03
Make Editable → 상/하단 분리
큐브를 편집 가능한 상태로 전환한 뒤, 위에서 두 번째 선을 기준으로 상단을 분리합니다.
더블클릭 (엣지 모드)
위에서 두 번째 선 루프 셀렉션
UF
Fill Selection — 선 기준 상단 면 전체 선택
UP
Split — 선택 영역을 별도 오브젝트로 분리
04
하단 오브젝트(원본) 윗면 제거
상단을 분리한 후 하단 오브젝트에 남아있는 위쪽 면을 삭제하면 상·하단이 깔끔하게 분리됩니다.
05
뚜껑 닫기 — Close Polygon Hole
분리 후 열린 구멍(뚜껑 없는 상태)을 닫아줍니다. 닫은 뒤 면을 4각형으로 정리해야 합니다.
MD
Close Polygon Hole — 열린 구멍을 면으로 닫음
MK
Line Cut — 뚜껑 면을 4각형으로 분할 정리
06
상단 캡 — 루즈 구멍 만들기
상단 오브젝트의 윗면에 Extrude Inner로 면을 줄인 뒤, Extrude로 면을 밀어 넣어 루즈가 나오는 통로를 만듭니다.
윗면 4개 선택 후 I
Extrude Inner — 면을 안쪽으로 줄이며 복사
D
Extrude — 면을 뽑아(깊이 추가) 홀 형태 완성
오늘 사용한 단축키 정리

UPSplit (오브젝트 분리)
MDClose Polygon Hole
MKLine Cut
IExtrude Inner
DExtrude
