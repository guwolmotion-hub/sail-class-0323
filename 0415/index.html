개념

키프레임(Keyframe)이란?

오브젝트(사진, 영상)에 움직임을 부여하는 AE의 핵심 기능입니다.

🎯
정의

원하는 시간에, 원하는 범위만큼 직접 움직임과 속도를 세팅하게 도와주는 기능.

⚙️
작동 원리

시작점과 끝점에 키프레임을 찍으면, 그 사이 움직임은 AE가 자동으로 계산해 등속운동하게 만드는 것이 기본 원리.

💎
키프레임 모양

타임라인에 마름모(◆) 아이콘으로 표시됩니다. 스톱워치 아이콘이 파란색으로 바뀌면 해당 속성의 키프레임이 활성화된 것.

단축키

컴포지션 & 프로젝트 단축키

작업 시작 전 자주 쓰는 기본 단축키입니다.

새 컴포지션 만들기

새 작업 공간 생성. 해상도·FPS·길이를 지정하는 가장 첫 번째 단계.

Ctrl+N
프로젝트 패널에 폴더 만들기

소스·Subcomp 등을 정리할 폴더 생성.

Ctrl+Shift+Alt+N
프로젝트 초기화

현재 프로젝트를 닫고 새 프로젝트 시작. 주의해서 사용.

Ctrl+Alt+N
레이어 복사

선택한 레이어를 바로 아래에 복제. Ctrl+C → Ctrl+V로도 가능.

Ctrl+D
선택 레이어의 키프레임만 보기

선택한 레이어에 찍힌 키프레임 속성만 타임라인에 펼쳐 보여줌. 작업 효율 최고.

U
Transform

Transform — 변형 5가지 속성

레이어를 선택하고 TRSPA 또는 레이어 패널의 Transform을 열면 확인할 수 있습니다.

01
Anchor Point
A

오브젝트 중심축의 위치. 회전·스케일의 기준점이 되므로 위치가 매우 중요.

02
Position
P

오브젝트의 화면 내 위치(X, Y 좌표). 이동 애니메이션의 핵심 속성.

03
Scale
S

오브젝트 크기(원본의 % 값). 비트맵 이미지는 임의로 키우면 화질 저하 주의!

⚠️ 비트맵 이미지 크기 임의 변경 금지
04
Rotation
R

오브젝트의 회전값. Anchor Point를 기준으로 회전. 360° = 1바퀴.

05
Opacity
T

오브젝트의 불투명도. 0% = 완전 투명, 100% = 완전 불투명. 페이드 인/아웃에 사용.

실습

키프레임 생성 순서

예시: 이미지가 0초에 왼쪽 끝에서 시작해 10초에 오른쪽 끝으로 이동하는 애니메이션 만들기

1
인디케이터를 시작 시간(0초)으로 이동

타임라인 상단의 재생헤드(인디케이터)를 시작점으로 드래그합니다.

2
오브젝트를 원하는 시작 위치로 이동

셀렉션 툴(V)로 이미지를 왼쪽 끝으로 드래그해 배치합니다.

3
스톱워치 아이콘 클릭 → 키프레임 활성화

Position 속성 옆 스톱워치(⏱) 아이콘을 클릭하면 파란색으로 변하며 타임라인에 ◆ 마름모가 생성됩니다.

4
인디케이터를 끝 시간(10초)으로 이동 후 위치 변경

인디케이터를 10초로 옮기고 오브젝트를 오른쪽 끝으로 드래그하면 자동으로 두 번째 키프레임이 생성됩니다.

여러 속성 동시에 키프레임 찍기

아래 애니메이션도 동일한 방법으로 각 속성에 스톱워치를 켜고 시작·끝 값을 세팅하면 됩니다.

Position왼쪽 끝 → 오른쪽 끝
Opacity0% → 100%
Scale100% → 200%
Rotation0° → 720° (2바퀴)
등속운동이 기본값 AE는 키프레임 사이를 일정한 속도(등속)로 자동 계산합니다. 자연스러운 가속/감속 표현을 원한다면 아래의 Easy Ease와 그래프 에디터가 필요합니다.
그래프 에디터

그래프 에디터 — 속도를 디자인한다

움직임이 있다고 끝이 아닙니다. 속도에 대비를 만들어 빠른 부분은 빠르게, 느린 부분은 확실하게 느리게 — 이것이 모션 그래픽 디자이너의 핵심 작업입니다.

Speed Graph
스피드 그래프

Y축 = 속도. 그래프가 올라갈수록 빠르고, 내려갈수록 느립니다. 경사가 급할수록 빠른 구간, 완만할수록 느린 구간.

Value Graph
밸류 그래프

Y축 = 위치(수치값). 이동 거리와 속도를 함께 표시. 경사가 급한 구간이 빠른 이동 구간.

📐

경사가 급할수록 빠르고, 완만할수록 느리다. 두 그래프 모두 동일한 원칙이 적용됩니다.

Easy Ease

Easy Ease — 자연스러운 가속/감속

키프레임을 선택하고 단축키를 누르면 등속운동을 자연스러운 가속/감속으로 즉시 변환합니다.

F9
Easy Ease

부드럽게 시작 → 중간에 빨라짐 → 끝에서 다시 부드럽게 감속. 가장 많이 쓰는 기본 설정.

그래프 조절 전 무조건 F9부터!
Shift+F9
Easy Ease In

초반은 빠르게 출발 → 끝에서 천천히 감속. 들어오는(In) 키프레임에 적용.

Ctrl+Shift+F9
Easy Ease Out

시작은 느리게 → 점점 가속해 빠르게 나감. 나가는(Out) 키프레임에 적용.

실무 원칙 기본적으로 F9 Easy Ease를 먼저 적용한 뒤 그래프 에디터로 세밀하게 조정하는 순서가 가장 효율적입니다. 빠른 부분은 확실히 빠르게, 느린 부분은 확실히 느리게 — 속도 대비가 클수록 모션이 살아납니다.
