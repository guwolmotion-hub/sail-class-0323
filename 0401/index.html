UV Mapping

UV 언랩 후 — 텍스처 작업을 어디서 하나?

C4D에서 UV를 펼친 뒤, 실제 텍스처를 그리고 입히는 작업은 외부 프로그램에서 진행합니다. 크게 두 가지 방법이 있으며, 작업 목적과 결과물의 복잡도에 따라 선택합니다.

🎨
방법 1 — Substance Painter
3D 포토샵
고급 텍스처링
3D 오브젝트 위에 직접 페인팅하듯 텍스처를 입힐 수 있는 전문 툴입니다. PBR 워크플로우(Roughness, Normal, Metallic 등)를 한 번에 작업할 수 있어 실사 퀄리티 텍스처 제작에 최적입니다.
C4D 모델링 완성 → FBX 또는 OBJ로 저장 → Substance Painter에서 작업
🖼
방법 2 — 포토샵 / 일러스트
2D 이미지 편집
평면 작업
C4D 안에서 UV 가이드를 PSD 파일로 추출한 뒤, 포토샵이나 일러스트레이터로 텍스처를 직접 그려 넣는 방법입니다. 로고, 라벨, 그래픽 텍스처 작업에 적합합니다.
UV 언랩 완료 → PSD 파일로 UV 가이드 추출 → 포토샵 / 일러스트로 작업
방법 1

Substance Painter — 3D 오브젝트에 직접 페인팅

Substance Painter(섭스턴스 페인터, 줄여서 "섭페")는 어도비가 인수한 3D 텍스처링 전문 툴입니다. 3D 뷰포트에서 오브젝트를 직접 보면서 브러쉬로 색상, 금속감, 거칠기 등을 동시에 칠할 수 있어 "3D 포토샵"이라고 부릅니다.

01
C4D에서 FBX 또는 OBJ로 내보내기
모델링이 완성되면 File → Export → FBX(.fbx) 또는 OBJ(.obj) 형식으로 저장합니다. 두 포맷 모두 Substance Painter가 지원하는 범용 3D 파일 형식입니다.
02
Substance Painter에 파일 불러오기
섭페를 실행하고 저장된 FBX/OBJ 파일을 열면 3D 오브젝트가 뷰포트에 로드됩니다. UV가 잘 펼쳐져 있어야 텍스처가 올바르게 배치됩니다.
03
PBR 채널별 텍스처 작업
Base Color, Roughness, Metallic, Normal 등 PBR 채널을 동시에 브러쉬로 작업합니다. 스마트 머티리얼, 필터, 제너레이터 등 강력한 기능을 제공합니다.
04
텍스처 Export → C4D/옥테인에 연결
작업이 완료되면 채널별 텍스처 이미지를 내보내고, C4D 또는 옥테인의 노드에 각각 연결하면 완성입니다.
방법 2

C4D → 포토샵 연동 — UV 가이드 추출 순서

C4D 내부의 BP-UV Edit 환경에서 UV 가이드(아웃라인)를 PSD 파일로 만든 뒤, 포토샵이나 일러스트에서 그 가이드 위에 직접 텍스처를 그리는 방법입니다. 로고·라벨·그래픽 텍스처 작업에 효율적입니다.

01
뉴 텍스처 파일 생성
UV 메뉴 → 파일 → 뉴 텍스처(New Texture)
파일명원하는 이름으로 설정
사이즈1024 / 2048 / 4096 px (해상도에 따라)
해상도72dpi (화면용) / 300dpi (인쇄용)
컬러 뎁스8bit 또는 16bit
02
UV 아웃라인 가이드 표시
UV 메뉴 → 레이어 → 뉴 레이어 생성 → 면 전부 선택(Ctrl+A) → 아웃라인 폴리곤즈(Outline Polygons) 클릭
이 단계에서 UV 전개도의 외곽선이 레이어에 그려집니다. 포토샵으로 가져갈 때 가이드 역할을 합니다.
03
바디페인트 패널 열기
메뉴 → 윈도우 → 바디페인트 3D → 레이어 매니저와 컬러 세팅 패널을 꺼내기
레이어 매니저는 포토샵의 레이어 패널과 동일한 역할을 합니다.
04
UV 위에 직접 페인팅 또는 포토샵으로 이동
UV 메뉴 → 페인트 → 브러쉬로 C4D 안에서 직접 그리거나, PSD 파일로 저장 후 포토샵/일러스트에서 UV 가이드 위에 텍스처를 작업합니다.
브러쉬 기능 등 대부분의 UI가 포토샵과 유사하므로 직관적으로 사용 가능합니다.

AI 이미지 생성

주요 AI 이미지 생성 툴 — 각자의 강점이 다르다

현재 AI 이미지 생성 툴은 빠르게 발전하고 있습니다. 각 툴마다 특화된 영역과 사용 방식이 다르므로, 목적에 따라 선택하는 것이 중요합니다. 크게 서버 방식(클라우드)과 로컬 방식으로 나뉩니다.

🎨
Midjourney
서버
이미지 생성 품질 1위. 예술적 감각이 뛰어난 결과물을 만들어냅니다. Discord 기반으로 운영되며, 모든 연산이 서버에서 처리됩니다.
이미지 수정(편집)은 비교적 약한 편
⚙
Comfy UI
로컬
Stable Diffusion 기반의 노드형 워크플로우 툴. 내 PC에서 직접 실행합니다. 고사양 GPU가 필요하지만 완전 무료이며, 커스터마이징과 자동화가 자유롭습니다.
생성 · 수정 모두 상위권, 단 진입 장벽 높음
✦
Gemini
서버
Google의 AI 모델. 이미지 수정(편집) 능력이 가장 뛰어납니다. Nanobanana 기능으로 세밀한 이미지 편집이 가능하며 구글 생태계와 통합됩니다.
이미지 수정 1위 — 생성보다 수정에 강함
🤖
ChatGPT (DALL·E)
서버
OpenAI의 ChatGPT에 내장된 이미지 생성 기능(DALL·E 기반). 텍스트 이해력이 좋아 복잡한 프롬프트도 정확하게 반영합니다. 가장 접근하기 쉬운 툴입니다.
범용성이 높고 사용 편의성 우수
🎬
Higgsfield
서버
영상·모션 생성에 특화된 AI 툴. 이미지 생성보다는 AI 비디오 생성 영역에서 두각을 나타내고 있습니다.
이미지보다 영상 생성에 적합

Comfy UI가 생성·수정 모두 상위권인 이유

Comfy UI는 Stable Diffusion이라는 오픈소스 모델을 기반으로 하며, 다양한 커스텀 모델과 워크플로우를 자유롭게 조합할 수 있습니다. 무료이면서 강력하지만, 진입 장벽(설치, 설정, GPU 요구)이 높은 것이 단점입니다.

AI 이해

AI 사이트 작동 원리 — 엔진과 UI는 다르다

우리가 사용하는 AI 이미지 사이트는 사실 두 가지 레이어로 나뉩니다. 이 구조를 이해하면 각 서비스의 가격 정책과 차이가 왜 생기는지 알 수 있습니다.

🧠
AI 모델 (엔진)
특정 회사 또는 나라에서 막대한 비용으로 개발한 핵심 AI 모델. 실제 이미지를 만드는 두뇌 역할.
GPT-4o Stable Diffusion Gemini Flux

엔진 위에 UI를 얹음
🖥
GUI (서비스 사이트)
엔진을 편하게 쓸 수 있도록 만든 인터페이스. 사용자는 이 UI를 통해 돈을 내고 엔진을 사용.

로컬 vs 서버 방식
☁
서버 방식 (Cloud)
  • AI 연산이 회사 서버에서 처리됨
  • 내 PC 사양 상관없이 사용 가능
  • 월정액 또는 크레딧 구매 필요
  • 인터넷 연결 필수
Midjourney ChatGPT Gemini
💻
로컬 방식 (Local)
  • AI 연산이 내 PC GPU에서 처리됨
  • 고사양 그래픽카드 필요 (VRAM 8GB+)
  • 사용 자체는 무료
  • 인터넷 없이도 사용 가능
Comfy UI Stable Diffusion WebUI
오픈소스 AI란?
Stable Diffusion처럼 모델의 코드와 가중치를 공개한 AI를 오픈소스 AI라고 합니다. 누구나 무료로 다운로드·수정·배포할 수 있어 Comfy UI 같은 다양한 커뮤니티 툴이 만들어집니다. 반면 GPT, Midjourney, Gemini 등은 코드를 공개하지 않는 클로즈드 소스 방식입니다.
