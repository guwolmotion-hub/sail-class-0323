시뮬레이션 바디

고스트바디 (Ghost Body)

다이나믹 시뮬레이션에서 가장 특이한 바디 타입 중 하나입니다. 실제로는 충돌하지 않지만, 충돌한 것처럼 다른 오브젝트에 영향을 줍니다.

일반 바디와 차이점

✗ 실제 물리 충돌 발생 안 함
✗ 중력의 영향을 받지 않음
✓ 다른 오브젝트에게 충돌로 인식시킴
✓ 트리거 기능과 결합해 특정 타이밍 제어 가능

트리거 / On Collision 와 함께 사용

고스트바디는 Trigger나 On Collision 기능과 함께 사용하면 강력해집니다.

  • 원하는 위치에서 시뮬레이션 시작
  • 원하는 타이밍에 애니메이션 발동
  • 충돌 없이 스타트 신호만 보낼 때 활용
MoGraph 이펙터

이펙터 (Effector)

클로너(Cloner)를 비롯한 MoGraph 기능에 추가적인 효과를 적용하는 도구입니다. 오브젝트들의 위치·회전·크기에 다양한 변화를 줄 수 있습니다.

Random Effector

PSR (Position / Scale / Rotation)에 랜덤성을 부여

  • 클로너로 복제된 오브젝트들이 각각 조금씩 다른 위치·크기·각도를 갖게 됨
  • 너무 규칙적으로 보이는 복제 배열을 자연스럽게 만들 때 필수
  • Seed 값으로 랜덤 패턴 변경 가능
Push Apart Effector

오브젝트들이 서로 겹치지 않게 자동으로 밀어냄

  • 복제 오브젝트가 서로 붙거나 겹칠 때 간격을 자동으로 벌려줌
  • Random Effector와 함께 사용하면 자연스러운 흩어진 배치 완성
  • Radius 값으로 최소 간격 조절
중력 제어

중력 개별 제어 — 오브젝트마다 다른 중력 설정

실습 목표: 위에서 작은 상자들이 쏟아져 내려올 때, 큰 상자는 공중에 떠 있으면서 형태만 찌그러지게 만들기.

구현 조건

✗
큰 상자는 중력 영향 없음

공중에 떠 있는 상태를 유지해야 하므로 중력 적용 불가

✗
큰 상자는 원래 형태로 복원되면 안 됨

소프트바디처럼 찌그러진 상태 그대로 유지

✓
작은 큐브들만 중력 981 적용

작은 상자들만 정상적으로 떨어지게

중력 개별 제어 방법

1
프로젝트 전체 중력 제거

Ctrl+D → 프로젝트 설정 → Dynamics 항목 → Gravity 값을 0으로 변경. 씬 전체의 중력이 사라짐.

Gravity: 981 → 0
2
작은 큐브에만 중력 개별 적용

작은 큐브 오브젝트의 시뮬레이션 태그 → Force 항목 → Force List 사용 또는 중력(Gravity) Force를 개별 추가해 981 값 설정

Attractor — 인력과 척력

↓↓
인력 (Attract)

오브젝트를 Attractor 방향으로 당기는 힘. 중심으로 모이게 하는 효과.

↑↑
척력 (Repel)

오브젝트를 Attractor에서 밀어내는 힘. 밖으로 퍼뜨리는 효과.

시뮬레이션 태그

포스 리스트 & 움직임 제어

시뮬레이션 태그의 Force 항목을 통해 각 오브젝트에 적용할 힘(포스)과 움직임 제한을 세밀하게 설정할 수 있습니다.

설정 항목
역할
Include Force List
포스 리스트 안에 있는 포스 만 이 오브젝트에 적용
Exclude Force List
포스 리스트 안에 있는 포스 만 이 오브젝트에 미적용 (나머지는 모두 적용)
Follow Position / Rotation
사이드브레이크 역할 — 오브젝트의 움직임을 완전히 고정
Linear Damping
브레이크 역할 — 이동 속도를 서서히 감속시켜 멈추게 함
Angular Damping
회전 브레이크 — 회전 속도를 서서히 감속시켜 멈추게 함
사이드브레이크 vs 브레이크 Follow Position/Rotation은 즉시 완전 고정(사이드브레이크), Linear/Angular Damping은 서서히 멈추는 감속(브레이크)입니다. 상황에 맞게 선택하세요.
물리 무게

Mass(무게) 설정 3가지

C4D 다이나믹에서 무게의 기준은 기본적으로 오브젝트의 크기입니다. 시뮬레이션 태그의 Mass 항목에서 세 가지 방식으로 설정할 수 있습니다.

World Density

크기 = 무게 (기본값)

오브젝트의 크기가 클수록 무겁고, 작을수록 가볍습니다. 수동 조정 불가.

큰 오브젝트
무거움 ↑
작은
가벼움 ↓
Custom Density

크기 기준 + 수치 조정 가능

크기에 따라 무게가 결정되지만, Density 수치를 조절해 원하는 무게로 변경 가능. 재질(밀도) 시뮬레이션에 적합.

Custom Mass

크기 무관 — 수치가 높은 쪽이 무거움

오브젝트 크기와 상관없이 입력한 Mass 수치가 높을수록 무겁게 작동합니다. 크기와 무게를 독립적으로 설정할 때 사용.

⚠️ 필수

시뮬레이션 베이크 (Bake) — 반드시 해야 하는 이유

시뮬레이션 작업 후 가장 중요한 마무리 단계입니다. 이걸 빠뜨리면 나중에 큰 낭패를 겪습니다.

베이크 안 하면 생기는 문제

🏫

학원에서 열심히 작업해 완성

→
🏠

집에서 파일 열었더니 시뮬레이션 결과가 달라짐

→
😱

학원 다시 와서 열었더니 또 다름

원인 시뮬레이션은 매번 계산을 새로 합니다. PC 성능, 랜덤 시드, 실행 환경에 따라 결과가 달라질 수 있습니다. 베이크하지 않으면 결과를 보장할 수 없습니다.

베이크 방법

1
시뮬레이션 재생 완료 확인

원하는 애니메이션 결과가 나왔는지 타임라인 끝까지 재생해 확인

2
시뮬레이션 베이크 실행

상단 메뉴 → Simulate → Bake All (또는 시뮬레이션 태그 우클릭 → Bake)

3
결과 확인

베이크 완료 후 타임라인을 앞뒤로 이동해도 동일한 결과가 유지되는지 확인
