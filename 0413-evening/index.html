BRDF 기초

BRDF Model — 빛을 얼마나 정교하게 계산할 것인가

BRDF(Bidirectional Reflectance Distribution Function)는 재질 계산 시 빛 에너지를 얼마나 물리적으로 정교하게 계산하느냐를 결정하는 모델입니다.

💡
빛이 표면에 닿았을 때 어느 방향으로, 얼마나 반사되는가?

같은 재질이라도 BRDF 모델에 따라 하이라이트 모양, 거칠기 표현, 빛번짐(Tail)의 양이 달라집니다.

⚙️
Specular BRDF와 Diffuse BRDF 두 가지로 나뉜다

Specular BRDF = 반사광(하이라이트)을 어떻게 계산할지 / Diffuse BRDF = 난반사(매트한 빛 확산)를 어떻게 계산할지

Specular BRDF

Specular BRDF 모델 비교

반사광(하이라이트)을 계산하는 방식입니다. 모델에 따라 하이라이트의 모양과 빛번짐(Tail)이 달라집니다.

Octane
단순 · 무난

Octane 전용 기본 모드. 빛을 가장 단순하게 계산하는 레거시 성향의 방식.

  • 무난하고 안정적인 스펙큘러 반사
  • 하이라이트가 부드럽고 자극적이지 않음
  • 물리적 정밀도는 GGX/Beckmann에 비해 낮음
  • 디테일한 거칠기 표현에는 한계 있음
일반 오브젝트 · 빠른 작업
GGX
PBR 표준 · 가장 현실적

물리 기반 렌더링(PBR)에서 산업 표준으로 사용되는 모델. 현실에 가장 가까운 반사 계산.

  • 러프니스가 높아질수록 Tail(빛번짐 꼬리)이 생성 → 실제 빛번짐과 가장 유사
  • 현실적이고 생동감 있는 하이라이트
  • 금속·코팅·자동차 도장 등 고급 재질에 최적
  • 러프니스 디테일이 중요한 컷에서 최강
금속 · 자동차 · 코팅 · 향수병
Beckmann
전통적 · 정돈된 느낌

전통적인 Microfacet BRDF. GGX보다 빛번짐이 적어 깔끔하고 차분한 하이라이트를 만든다.

  • 하이라이트가 깔끔하고 정돈된 느낌
  • 러프니스가 높아져도 광택이 차분하게 유지
  • GGX보다 Tail(빛번짐 꼬리)이 훨씬 적음
  • 과한 반사 없이 제품 광택 표현에 적합
플라스틱 · 제품 컷 · 스타일라이즈
Ward
애니소트로피 전용

방향성 거칠기(Anisotropy)를 표현하는 특수 모델. 결이 있는 금속 재질에 특화.

  • 애니소트로피 — 방향에 따라 반사 특성이 다름
  • 헤어라인 금속, 브러시드 알루미늄, 다마스커스 패턴
  • 결 방향이 있는 재질에만 사용
브러시드 메탈 · 헤어라인 · 다마스커스
AO (Ambient Occlusion) 음영을 추가해 오브젝트에 깊이감을 부여하는 패스. 미세한 틈이나 접힘 부분의 어두운 음영을 계산해 리얼리티를 높입니다.

Diffuse BRDF

Diffuse BRDF 모델 — 난반사 계산 방식

빛이 표면에서 매트하게 퍼지는 난반사를 어떻게 계산할지 결정합니다. 고유색(Diffuse) 채널의 표현 품질에 영향을 줍니다.

Octane

Specular BRDF의 Octane 모드와 동일한 방식. 단순하게 계산하는 무난한 기본 모드.

단순 계산 무난한 기본값
Lambertian

빛이 표면에 균일하게 퍼지는 가장 기본적인 난반사 모델.

  • 계산이 단순하고 안정적
  • 매트한 느낌이지만 아주 현실적이진 않음
  • 표면 거칠기·미세 난반사가 단순화됨
  • 무광 재질에는 장점 → 단, 노말/범프 왜곡 시 약간 평평해 보일 수 있음
무광 오브젝트 단순 소재
Oren-Nayar

표면이 거칠수록 빛이 더 복잡하게 퍼지는 것을 시뮬레이션. 현실에 가장 가까운 Diffuse 모델.

  • 거칠기에 따라 빛 확산이 자연스럽게 변함
  • 각도에 따른 음영 변화가 자연스러움
  • 범프·노말이 들어간 디테일한 표현에 유리
  • 거친 벽, 천, 분필, 종이, 콘크리트, 피부 등에 최적
벽 · 천 · 피부 콘크리트 · 종이

Lambertian vs Oren-Nayar 한눈 비교

Lambertian
  • 균일한 빛 확산
  • 단순·빠른 계산
  • 완전 무광 소재에 적합
  • 범프 왜곡시 약간 평평하게 보일 수 있음
VS
Oren-Nayar
  • 거칠기에 따른 복잡한 빛 확산
  • 물리적으로 정교한 계산
  • 거친 질감 소재에 최적
  • 범프·노말 디테일 표현력 우수
정리

BRDF 모델 전체 요약

모델 선택의 핵심 기준을 정리했습니다.

GGX

물리적으로 가장 현실적. 금속·코팅·자동차. Tail 빛번짐이 실제와 동일.

현실적 렌더
Beckmann

차분하고 정돈된 하이라이트. 플라스틱·스타일라이즈. 빛번짐 최소.

깔끔한 광택
Ward

방향성 거칠기 전용. 브러시드 메탈·헤어라인·다마스커스.

애니소트로피
Oren-Nayar

Diffuse 난반사 현실 모델. 거친 벽·천·피부·콘크리트.

자연스러운 매트
선택 원칙 현실적인 재질 → GGX / 깔끔한 제품 → Beckmann / 결 있는 금속 → Ward / 거친 표면 Diffuse → Oren-Nayar. 모를 때는 GGX가 가장 안전한 선택입니다.
