렌더모드 심화

Photon Tracing & PMC — 특수 목적 렌더모드

지난 시간에 배운 4가지 렌더모드 중 Photon Tracing과 PMC를 더 깊이 살펴봅니다. 두 모드 모두 특수한 상황에서 쓰이며, 범용 작업에는 Path Tracing이 기본입니다.

03
Photon Tracing
포톤트레이싱
유리 특화
Path Tracing → 업그레이드 버전
Path Tracing을 기반으로 유리의 커스틱스(Caustics) 같은 복잡한 빛 굴절·투과 현상을 더욱 정교하게 표현하도록 발전된 렌더 기능입니다.
⚠ 완벽하지는 않습니다. 유리·투명 재질이 핵심인 씬에서 Path Tracing보다 나은 결과를 기대하는 정도로 활용합니다.
커스틱스(Caustics)란?
빛이 유리나 물을 통과할 때 바닥이나 벽에 생기는 특유의 밝은 빛 패턴입니다. 유리잔 밑에 생기는 반짝이는 빛 무늬를 떠올려보세요.
04
PMC
퍼뮤테이션 몬테카를로
극사실주의
사실상 가장 물리적으로 현실과 비슷하게 렌더링되는 모드입니다. 빛의 경로를 몬테카를로 방식으로 반복 계산해 극도로 정밀한 결과를 냅니다.
단점 렌더링 시간이 굉장히 오래 걸림
장점 현실보다 더 현실 같은 극사실주의 표현 가능
주요 사용 상황
자동차 광고영상처럼 "실제보다 더 실제 같아 보여야 한다"는 수준의 극한 퀄리티가 요구되는 작업. 일반 프로젝트에서는 거의 사용하지 않습니다.

핵심 원리

옥테인 = 언바이어스드 렌더 — 계산 횟수가 전부다

옥테인은 언바이어스드(Unbiased) 렌더러입니다. 즉, 노이즈 방식으로 계산하며 계산 횟수(샘플)가 많아질수록 노이즈가 서서히 줄어들고 이미지가 깨끗해집니다. 절대적으로 완벽한 수치는 없습니다 — 상황에 따라 타협하며 결정합니다.

노이즈 타협

노이즈와 타협하는 법 — Adaptive Sampling 활용

렌더링 특성상 노이즈를 완전히 없애는 것은 불가능합니다. 무한히 샘플을 올릴 수도 없으므로, 적절한 지점에서 타협하고 나머지 노이즈는 후반작업에서 제거하는 방식을 사용합니다.

세밀 설정

Depth — 채널별 세분화 계산 횟수

Depth는 샘플보다 더 세분화된 계산 횟수입니다. 색상, 반사, 빛 투과(스페큘러) 등 각 채널별로 얼마나 깊이 계산할지를 개별 설정합니다. 역시 절대값은 없고 눈으로 확인하며 결정합니다.

🎨
색상 (Diffuse Depth)
오브젝트의 기본 색상이 여러 표면을 통해 간접적으로 반사되는 횟수를 제어합니다.
🪞
반사 (Specular Depth)
거울·금속 등 반사 재질에서 빛이 반사되는 횟수를 계산합니다. 복잡한 반사 환경일수록 높은 값이 필요합니다.
💡
빛 투과 (Transmission Depth)
유리·물 등 투명 재질을 빛이 통과하는 횟수를 계산합니다. 여러 겹의 유리가 있을 때 높은 값이 필요합니다.
