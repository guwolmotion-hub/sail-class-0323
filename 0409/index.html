메터리얼 레이어 — 펄 글로시 질감 만들기

레이어 형태로 여러 층을 쌓아가며 재질을 표현하는 기능입니다. 오늘의 목표: 펄이 들어간 글로시 질감 제작.

핵심 개념 메터리얼 레이어는 포토샵의 레이어처럼 재질을 겹겹이 쌓습니다. 아래부터 Diffuse → 펄(Pearl) → Specular 순으로 쌓으면 됩니다.

펄 글로시 재질 제작 순서

1
메터리얼 레이어 활성화

재질 노드 클릭 → Basic 항목에서 메터리얼 레이어 체크박스 활성화

2
레이어 그룹 추가

메터리얼 레이어 항목 → Add 버튼 → 레이어 그룹 선택. 여러 층을 묶어 관리할 때 사용하는 컨테이너

3
메탈릭 레이어 + Flake 노드로 펄 층 제작

메탈릭 레이어를 추가하고 Flake 노드를 연결. 크기와 맵핑 상태 조정 후 Fall Off 노드로 아래 Diffuse 층과 자연스럽게 블렌딩

4
스펙큘러 레이어 추가

마지막으로 스펙큘러(반사) 레이어를 추가해 광택층 완성

Fall Off 노드

오브젝트의 각도에 따라 효과 강도를 조절하는 노드입니다.

  • 카메라를 향하는 면 → 강하게 표현
  • 측면·뒷면 → 점점 약해짐 (또는 반대)
  • 프레넬(Fresnel) 효과 연출에 필수
Flake 노드

도료 속 반짝이는 금속 파편(플레이크)을 표현하는 노드입니다.

  • 자동차 도장·펄 재질에 핵심
  • Scale로 파편 크기 조절
  • 맵핑 조정으로 밀도·패턴 변경

로고 작업

오브젝트에 로고 넣기

특정 면에만 로고 재질을 적용하고 위치와 타일 현상을 제어하는 방법입니다.

로고 작업 순서

1
로고용 재질 준비 & 면 선택 적용

새 재질(글로시 or 메탈릭) 생성 → 로고 이미지를 컬러/알파 채널에 연결 → 오브젝트에서 로고를 넣을 면만 선택 후 재질 드래그 적용

로고 이미지 권장 형식 PNG 알파 배경 검정 + 로고 흰색
2
프로젝션 모드 → Flat으로 변경

재질 태그 클릭 → 프로젝션(Projection) 항목을 Flat으로 변경. 텍스처 모드(좌측 버튼)로 크기·위치 직접 조절

3
타일 현상 제거

로고 이미지가 반복되는 타일 현상이 생길 경우 아래 두 방법 중 선택:

방법 A

재질 태그 → 타일(Tile) 옵션 비활성화

방법 B

이미지 텍스처 노드 → Border Mode UV → Black으로 변경

4
재질 이름 정리

완성된 재질들은 역할에 맞게 이름을 명확하게 변경. 재질이 많아질수록 이름 관리가 핵심!

Flat 프로젝션이란? 텍스처를 정면에서 납작하게 투영하는 방식입니다. 로고·스티커처럼 특정 방향에서만 보이는 이미지를 붙일 때 적합합니다.
최종 점검

최종 렌더 전 옥테인 가렌더

편집에 들어가기 전, 옥테인이 적용된 상태로 한 번 더 가렌더를 돌려 전체 흐름을 확인합니다.

⚡

샘플 수 낮게 설정

가렌더 목적은 품질 확인이 아닌 흐름 확인입니다. 샘플 수를 최대한 낮게 설정해 렌더 속도를 높이세요.

✅

확인 항목

  • 재질·조명이 의도대로 보이는지
  • 카메라 구도 및 움직임
  • 애니메이션 타이밍
  • 전체 씬 길이와 흐름
작업 순서 가렌더 확인 → 수정 → 최종 렌더. 최종 렌더는 한 번만! 가렌더에서 충분히 검토하면 불필요한 재렌더를 줄일 수 있습니다.
영상 기획

영상 기획 — 툴보다 기획이 먼저다

화려한 영상은 많습니다. 하지만 화려하다고 다 좋은 것이 아닙니다. 영상의 목적을 먼저 정의해야 합니다.

🎯

영상이라는 장르의 목적

짧은 시간 안에 정확한 정보를 상대에게 전달하는 것

긴 글을 한 장의 이미지로, 복잡한 메시지를 몇 초의 영상으로 — 이것이 영상의 본질입니다.

인강의 함정 인강으로 스킬을 배운 학생들의 포트폴리오는 구도·연출·스타일이 거의 똑같습니다. 소스만 다를 뿐 기획 능력이 없으면 현장에서 3개월을 버티기 어렵습니다. 기본기와 기획력은 본인이 직접 키워야 합니다.
