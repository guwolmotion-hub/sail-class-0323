기본 기능

Extrude — 면과 선을 입체감 있게 돌출시키는 기능

Extrude(익스트루드)는 3D 모델링에서 가장 자주 쓰이는 핵심 기능 중 하나입니다. 선택한 면(Polygon)이나 선(Edge)을 밖으로 뽑아내거나(돌출), 안으로 밀어넣어 새로운 형태를 만들 수 있습니다.

↑
면 뽑기 (Outward)
선택한 면을 바깥 방향으로 돌출시켜 튀어나온 형태를 만듭니다. 버튼, 손잡이, 돌기 등을 만들 때 활용합니다.
↓
면 집어넣기 (Inward)
선택한 면을 안쪽으로 밀어넣어 홈, 구멍, 오목한 부분을 만듭니다. 나사 구멍, 홈, 요철 등에 활용합니다.
→
스플라인 면 변환
2D 선(스플라인)에 두께를 주어 3D 면으로 변환합니다. 이전 수업의 스튜디오 모델링이 바로 이 방식입니다.
⚠
시메트리(Symmetry)와 함께 사용할 때 반드시 주의!

시메트리는 한쪽을 복사해 반대편을 자동 생성합니다. 이 과정에서 안 보이는 내부나 이음새 부분에 불필요한 뚜껑(Cap)이나 면(Polygon)이 생기는 경우가 있습니다. 이 잉여 면들은 지금 당장은 눈에 보이지 않아 모르고 지나치기 쉽지만, 나중에 텍스처 매핑이나 이펙트(효과)를 적용할 때 오류의 원인이 됩니다.

✗ 시메트리 적용 후 안 보이는 면을 그냥 방치 → 매핑/이펙트 오류 발생 위험
✓ 시메트리 사용 후 복사된 부분의 면 상태를 반드시 직접 눈으로 확인하고 정리
선생님 이야기

영상·디자인 업계 현실 — 툴러 vs 디자이너

기술을 배우는 것과 함께, 자신이 어떤 방향으로 성장할지도 생각해봐야 합니다. 업계에서 살아남는 사람과 대체되는 사람의 차이는 기술의 숙련도가 아니라 생각하고 기획하는 능력에 있습니다.

🤖
툴러 (Tool User)
"시키는 대로만 하는 사람"
  • 주어진 작업을 정해진 방식대로 반복 실행
  • 기획·방향 결정 없이 기술만 사용
  • AI가 동일한 작업을 더 빠르게 수행 가능
  • AI 기술 발전에 따라 대체될 가능성 높음
AI 대체 위협 높음
VS
🎨
디자이너 (Creator)
"기획 · 디자인 · 모션까지 스스로 하는 사람"
  • 콘셉트 기획부터 최종 결과물까지 주도
  • 도구를 수단으로 활용해 자신만의 작품 창작
  • AI를 도구로 활용해 오히려 효율 극대화
  • 개성과 창의성으로 대체 불가능한 가치 창출
AI 시대에도 성장 가능
★
스페셜리스트급으로 파지 않아도 퀄리티를 높게 만들 수 있다

모든 분야를 깊이 파는 것보다, 전반적인 흐름을 이해하고 기획·디자인·모션을 스스로 연결할 수 있는 능력이 중요합니다. 도구의 노예가 되지 말고, 도구를 활용하는 사람이 되세요.

UV Mapping 기초

UV 매핑 — 컴퓨터는 바보, 시킨 것밖에 못 한다

3D 오브젝트에 이미지 텍스처를 올바르게 입히려면 UV 매핑을 이해해야 합니다. 컴퓨터는 스스로 판단하지 못합니다. "이 오브젝트에 어떻게 이미지를 펼칠지"를 우리가 직접 정확하게 알려줘야 합니다.

🖥
컴퓨터의 기본 동작 원리

모든 오브젝트는 박스(Box) 형태라고 생각하고 전개도를 펼칩니다. 구(Sphere)도, 실린더도, 복잡한 캐릭터도 결국 UV 좌표는 납작한 2D 평면에 펼쳐진 전개도 형태가 됩니다. 우리가 그 전개도를 얼마나 깔끔하게 펼치느냐가 UV 매핑의 핵심입니다.

01
3D 오브젝트를 2D로 "펼친다"
종이로 만든 상자를 가위로 잘라 납작하게 펼치면 전개도가 되듯, 3D 메쉬도 적절한 위치에 "절취선(Seam)"을 넣어 펼치면 평면 전개도(UV)가 됩니다.
02
전개도 위에 이미지를 올린다
잘 펼쳐진 UV 전개도 위에 텍스처 이미지를 올려놓으면, 3D 오브젝트에는 그 이미지가 왜곡 없이 자연스럽게 입혀집니다.
03
절취선(Seam)이 핵심이다
어디를 잘라서 펼칠지 결정하는 것이 UV 매핑의 핵심 작업입니다. 잘못 자르면 전개도가 뒤틀리거나 겹쳐서 텍스처가 이상하게 표시됩니다.
핵심 개념

Seam (심라인) — 무조건 생길 수밖에 없는 재봉선

Seam(심, 심라인)은 3D 오브젝트를 2D로 펼치기 위해 표면을 자르는 절취선입니다. 옷의 재봉선(솔기)과 같은 개념입니다. 핵심은 — 심라인은 반드시 생깁니다. 없앨 수 없으니, 최대한 보이지 않는 곳에 숨기는 전략이 필요합니다.

Seam이란 무엇인가?
👕
옷의 재봉선
옷감을 이어 붙인 솔기. 보통 겨드랑이, 허리 등 잘 안 보이는 곳에 위치.
=
🗺
UV의 심라인
3D 메쉬를 2D로 펼치기 위한 절취선. 렌더링에서 잘 보이지 않는 곳에 위치시켜야 함.
!
심라인은 무조건 생긴다
어떤 3D 오브젝트든 완벽하게 심(이음매) 없이 2D로 펼치는 것은 불가능합니다. 지구본을 평면 지도로 만들 때 반드시 어딘가가 잘리거나 왜곡되는 것과 같은 원리입니다.
★
전략: 최대한 안 보이는 부분에 위치시키기
심라인이 생길 수밖에 없다면, 카메라에 잡히지 않는 곳, 뒷면, 아랫면, 안쪽 등 눈에 띄지 않는 위치를 절취선으로 선택하는 것이 UV 매핑의 핵심 전략입니다.
✓
기본 공식: 뚜껑 따고 → 뒷부분 절취선 잡기
일반적인 오브젝트의 기본 UV 매핑 방법입니다. 먼저 앞뒤 뚜껑(Cap)을 분리하고, 몸통의 안 보이는 뒷부분 라인을 절취선으로 잡아 몸통을 펼치면 깔끔한 전개도를 만들 수 있습니다.
실습 순서

UV 언랩 전체 작업 순서

심라인을 잡고 UV를 펼치는 전체 과정입니다. 특히 선 선택을 셀렉션 태그로 저장하는 습관을 들이는 것이 실수를 줄이는 핵심입니다.

01
심라인으로 쓸 선(Edge) 선택
Edge 모드에서 절취선으로 사용할 선을 선택합니다. 뚜껑 분리선, 몸통 뒷면 라인 등 전략적으로 위치를 고려해 선택하세요.
02
선택한 선을 셀렉션 태그로 저장
선 선택 상태에서 실수로 클릭해 선택이 해제될 수 있기 때문에 반드시 셀렉션 태그로 저장해 둬야 합니다.
V → Select → Set Selection = 셀렉션 태그로 저장됨
03
BP-UV Edit 패널 열기
우측 상단 레이아웃 패널에서 BP-UV Edit를 선택합니다. UV 편집 전용 작업 환경으로 전환됩니다.
04
기존 UV 태그 초기화
UV 에디트 창에서 기존에 있던 UV 태그를 먼저 초기화합니다.
Set UV from Projection
05
선 선택 상태에서 UV Unwrap 실행
저장해 둔 셀렉션 태그(심라인)가 선택된 상태에서 UV Unwrap을 실행하면 자동으로 오브젝트가 펼쳐집니다.
06
결과 확인 및 겹침 수동 수정
덩어리들이 올바르게 분리됐는지 확인합니다. 꼬다리(끝 부분)가 겹쳐 있는 경우 수동으로 분리 작업을 해야 합니다.
이건 내가 잘못한 게 아니라 "컴퓨터가 바보"이기 때문입니다. 당황하지 말고 수동으로 분리해주세요.
실습 예시

킷캣 오브젝트 UV 언랩 — 눕힌 실린더 형태

킷캣(KitKat) 형태는 옆으로 눕힌 실린더와 비슷한 구조입니다. 좌우 양끝(뚜껑)과 긴 몸통(옆면)을 분리해서 펼치는 방식으로 언랩합니다.

킷캣 오브젝트 구조
왼쪽
뚜껑
몸통 (옆면)
▲ 아래쪽 라인 → 심라인
오른쪽
뚜껑
1단계
뚜껑 먼저 분리
좌우 양쪽 뚜껑(Cap)의 가장자리 선을 심라인으로 잡아 몸통과 분리합니다. 뚜껑은 별도의 UV 덩어리로 분리됩니다.
뚜껑의 기준점(Seam 위치)은 나중에 이미지를 어느 부분에 넣을지에 따라 달라집니다. 로고나 텍스트를 넣을 위치를 미리 생각하고 심라인을 결정하세요.
2단계
몸통 아래쪽 라인에 심라인 잡기
몸통(옆면)은 아래쪽 라인을 심라인으로 잡아 펼칩니다. 아랫면은 킷캣이 놓였을 때 바닥에 가려져 보이지 않으므로, 심라인이 있어도 렌더링에서 눈에 띄지 않습니다.
결과
5개 덩어리 확인
언랩이 완료되면 UV 에디터에 5개의 UV 덩어리가 나타나야 합니다. (왼쪽 뚜껑 / 오른쪽 뚜껑 / 몸통 앞면 / 몸통 뒷면 / 기타)
덩어리 수가 맞지 않거나 겹쳐 있다면 심라인 설정을 다시 확인하거나, 겹친 덩어리를 수동으로 분리합니다.
💡
꼬다리가 겹쳐도 당황하지 마세요

언랩 후 끝 부분(꼬다리)이 겹쳐 있는 경우가 종종 생깁니다. 이건 내가 잘못한 게 아닙니다. UV 언랩 알고리즘이 완벽하지 않기 때문입니다. 겹친 부분을 UV 에디터에서 수동으로 이동·분리해주면 됩니다.
