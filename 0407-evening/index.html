스크래치 이미지 합성으로 표현하기

몬스터볼 하단 메탈릭 재질의 포인트 — 단 하나의 이미지로 완벽한 스크래치를 찾기는 거의 불가능합니다. 여러 이미지를 합쳐 원하는 모양을 직접 만드는 것이 핵심입니다.

Tip! 마음에 드는 스크래치 이미지 하나를 찾으려 하지 말고, 여러 이미지를 조합해 원하는 모양을 만드는 것이 훨씬 효율적입니다.
🖼️

Add 블렌드 모드로 합성

스크래치·더트·그런지 이미지는 흰색 영역이 핵심입니다. Add 기능을 사용하면 각 이미지의 밝은 부분(스크래치)만 누적되어 자연스럽게 합쳐집니다.

블렌드 모드: Add
→ 밝은 영역(스크래치)이 합산됨
🔄

이미지 크기와 맵핑의 중요성

같은 이미지라도 맵핑 상태나 이미지 크기에 따라 완전히 다른 텍스처처럼 보일 수 있습니다. 크기(Scale) 조절을 적극 활용하세요.

  • 작게 → 잔 스크래치 느낌
  • 크게 → 거칠고 굵은 스크래치
  • 회전값 변경 → 방향감 다양화

스크래치 합성 워크플로우

1
소스 이미지 수집

Scratch / Dirt / Grunge 텍스처 2~3장 준비 (흰색 스크래치, 검은 배경 형태)

2
Material Layer에 추가

각 이미지를 별도 레이어로 추가, 블렌드 모드를 Add로 설정

3
크기와 오프셋 조절

각 레이어의 UV Scale을 다르게 설정해 패턴이 겹쳐 보이지 않도록 조정

4
전체 강도 조절

Dirt 채널이나 레이어 불투명도로 스크래치 강도 최종 조정

재질 파이프라인

재질 표현 도구의 위계

C4D에서 재질을 표현하는 방법에는 여러 단계가 있습니다. 각 도구의 목적과 적합한 상황을 이해하면 작업 흐름이 훨씬 효율적입니다.

Level 1
Substance (서브스탠스)

PBR 기반 실사 재질 전문 툴. 가장 강력하고 정교한 재질 표현 가능

이상적
Level 2
Octane (옥테인 렌더러)

물리 기반 렌더링 내장 노드로 복잡한 재질 구현 가능. 서브스탠스를 못할 때 차선책

차선책
Level 3
Photoshop (포토샵)

텍스처 이미지를 직접 편집·합성. 위 두 도구를 사용하지 못할 때의 대안

대안
핵심 메시지 도구가 없거나 못 쓰는 상황에 맞춰 유연하게 대응하는 것이 실무 역량입니다. 포토샵만으로도 충분히 좋은 재질 표현이 가능합니다.
애니메이션

소프트바디 바운스 애니메이션

몬스터볼이 바닥에 통~ 튀는 말랑한 느낌의 애니메이션. 하지만 우리가 모델링한 몬스터볼에는 소프트바디를 직접 적용하기 어려운 이유가 있습니다.

문제 — 현재 몬스터볼의 상태

✗
오브젝트가 분리되어 있다

상단·하단·라인·버튼 등 여러 오브젝트로 나뉘어져 있어 소프트바디 적용 대상이 불명확

✗
삼각형 폴리곤이 있다

소프트바디는 쿼드(4각형) 폴리곤에서 안정적으로 작동. 삼각형이 있으면 변형이 예측 불가

✗
기본 면이 너무 많다

폴리곤 수가 많을수록 시뮬레이션 계산이 무거워지고 불안정해짐

Q. 그렇다면 이런 오브젝트는 소프트바디를 쓸 수 없는가?  A. 아니요!  메쉬 디포머를 활용하면 가능합니다.
Mesh Deformer

메쉬 디포머로 소프트바디 우회하기

직접 소프트바디를 걸 수 없는 복잡한 오브젝트에 시뮬레이션 효과를 적용하는 전문 테크닉입니다.

메쉬 디포머란?

케이지(Cage) 오브젝트 안에 들어있는 대상 오브젝트의 움직임과 형태 변화를 제어하는 디포머입니다. 케이지가 변형되면 내부 오브젝트도 따라서 변형됩니다.

Another Object (어나더 오브젝트)
Mesh Deformer (메쉬 디포머)
심플한 오브젝트가 시뮬레이션을 받고,
복잡한 오브젝트가 그것을 따라감
케이지 오브젝트가 소프트바디를 받고,
메쉬 디포머가 케이지 변형을 내부 오브젝트에 전달
소프트바디와 호환 ✗
소프트바디와 호환 ✓

메쉬 디포머 설정 순서

1
케이지 오브젝트 만들기

몬스터볼과 비슷한 크기·모양의 단순한 구(Sphere) 생성. 면 수는 최대한 적게 (예: 8×8 정도)

2
케이지에 소프트바디 태그 적용

케이지 구에 Soft Body 태그를 걸고 시뮬레이션 파라미터 설정

3
몬스터볼에 메쉬 디포머 추가

몬스터볼 Null 하위에 Mesh Deformer 추가 (Deformer 메뉴)

4
케이지 연결

메쉬 디포머의 Cage 슬롯에 케이지 구를 드래그 연결 → 초기화(Initialize)

5
결과 확인

재생 시 케이지가 소프트바디로 변형되면, 몬스터볼도 자연스럽게 따라 변형됨

핵심 원리 못생긴 면(삼각형, 복잡한 폴리곤)을 가진 오브젝트도 깔끔한 케이지만 있으면 마치 소프트바디가 정상적으로 걸린 것처럼 보이게 할 수 있습니다.
재질 비교

믹스 메터리얼 vs 컴포지트 메터리얼

두 메터리얼 모두 2개 이상의 재질을 섞을 때 사용합니다. 하지만 동작 방식과 장단점이 다릅니다.

Mix Material (믹스)

매터리얼 매니저 안에 있는 재질들을 섞는 메터리얼

장점

  • 재료 재질을 언제든 재활용 가능
  • 다른 믹스 메터리얼에서도 같은 재질 공유

단점

  • 섞을 때마다 재질이 매니저에 추가 → 관리가 복잡
  • 공유 재질 수정 시 해당 재질을 쓰는 모든 믹스가 함께 변경
  • 하나의 결과물만 수정하기 어려움

Composite Material (컴포지트)

매터리얼 매니저에 표시되지 않는 서브 재질로 구성

장점

  • 매니저가 복잡해지지 않아 정리가 깔끔
  • 각 컴포지트가 독립적 → 개별 수정 용이

단점

  • 서브 재질이 매니저에 없어서 재사용 불가
  • 정성 들여 만들어도 다른 오브젝트에 쓰려면 새로 제작

선택 가이드

믹스 메터리얼 사용 시

여러 오브젝트에 걸쳐 같은 베이스 재질을 공유해서 섞어야 할 때 (예: 공통 금속 재질 + 각자 다른 오버레이)

컴포지트 메터리얼 사용 시

한 오브젝트에만 쓸 복잡한 레이어 재질을 만들 때 (예: 몬스터볼 하단의 스크래치·더트·그런지 조합)
