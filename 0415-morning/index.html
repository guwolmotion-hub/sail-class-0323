핵심 설정

Intermediate Points — 스플라인 문제의 99% 해결책

스플라인 관련 애니메이션 작업 시 생기는 대부분의 문제는 Intermediate Points 설정을 바꾸면 해결됩니다. 기본값 Adaptive에서 Uniform 또는 Natural로 변경해보세요.

자주 발생하는 증상 공이 곡선 경로를 따라가다 속도가 들쑥날쑥하거나, 특정 구간에서 튀거나 멈추는 현상 → Intermediate Points 확인!
Adaptive 기본값

직선 구간은 포인트를 적게, 곡선 구간은 포인트를 많게 자동 배치. 포인트 수가 불균일해 애니메이션 속도가 튀는 원인이 됨.

⚠️ 애니메이션 작업 시 속도 튐 발생 가능
Uniform 애니메이션 추천

직선·곡선 구분 없이 일정한 간격으로 포인트 배치. 등간격이라 애니메이션 속도가 일정하게 유지됨.

✓ 스플라인 애니메이션 문제 해결의 핵심
Natural 모델링 추천

직선에도 곡선에도 적당한 수의 포인트를 배치. 정상적인 면 정리 시 가장 깔끔한 결과물을 만드는 모드.

✓ 스플라인 모델링 면 정리에 최적
Subdivide 과도한 분할

구분 없이 무조건 많은 포인트를 생성. 파일이 무거워질 수 있어 특수한 경우 외에는 잘 사용하지 않음.

⚠️ 불필요하게 무거워질 수 있음
Xpresso 심화

곡선 경로 공굴리기 — Align to Spline 연동

직선이 아닌 곡선 경로를 따라 공이 자연스럽게 굴러가게 하는 것이 목표입니다. Align to Spline의 Position과 공의 Rotation을 Xpresso로 연동합니다.

🎯 작업 목표

Align to Spline의 Position 0~100%가 움직일 때, 공의 Rotation B가 경로 길이에 비례해 자연스럽게 회전하게 만들기

①
공의 둘레 길이 계산

공이 자신의 둘레(2πr)만큼 이동하면 정확히 한 바퀴가 회전. 반지름으로 총 회전수 계산 가능.

②
경로 스플라인의 전체 길이 확인

공이 이동하는 총 거리 = 스플라인의 전체 길이. 이 값을 Xpresso로 가져와야 한다.

③
총 이동 거리 ÷ 둘레 = 총 회전수

스플라인 길이를 공의 둘레로 나누면 0~100% 이동 시 총 몇 바퀴 돌지 계산됨. 이 값을 Range Mapper로 Rotation B에 매핑.

💡
스플라인 애니메이션 문제가 생기면 먼저 확인

공이 경로를 따라 이동할 때 속도가 튀거나 불규칙하다면 → 스플라인의 Intermediate Points를 Uniform으로 변경!

스플라인 모델링

스플라인 → 메쉬 변환 6가지

스플라인(선)은 자체로는 보이지 않습니다. 아래 제너레이터를 사용해 3D 메쉬(면)로 변환합니다. 로고 모델링, 원통형 모델링에 핵심적으로 사용됩니다.

⬆️

Extrude

선 모양대로 오브젝트를 돌출시켜 메쉬화. 2D 로고·텍스트를 3D로 만들 때 가장 많이 사용.

로고 3D화, 텍스트 3D
🔄

Lathe

스플라인을 한 바퀴 회전시켜 메쉬화. 병, 컵, 항아리 같은 회전 대칭 형태에 최적.

병, 컵, 항아리, 원통형
🚇

Sweep

경로(Path) 스플라인을 따라 단면 스플라인이 이동하며 터널 형태의 메쉬를 생성.

파이프, 튜브, 전선, 로프
🔀

Loft

여러 스플라인 단면을 순서대로 연결해 메쉬화. 일러스트의 블렌드 기능과 유사.

날개, 배, 유기적 형태
✏️

Vector Lizer

흑백 이미지를 가져와서 자동으로 스플라인 선으로 변환. 벡터화 기능.

이미지 → 스플라인 변환
✂️

Spline Mask

여러 스플라인을 합치거나 뺄 때 사용. 일러스트의 패스파인더와 동일한 개념.

스플라인 합치기·빼기·교차
모델링 실습

아이폰 모델링 — 8단계 작업 순서

Extrude를 활용한 스플라인 기반 아이폰 모델링입니다. 도면(레퍼런스 이미지)을 각 뷰에 배치하고 순서에 따라 진행합니다.

1
도면 이미지를 각 뷰에 배치

Top, Front, Right 뷰에 아이폰 도면(정면·측면·상면) 이미지를 각각 배치해 모델링 기준으로 활용.

2
Rectangle 스플라인으로 크기와 라운딩 맞추기

Rectangle Spline을 꺼내 아이폰 외형 크기에 맞게 조정. 모서리 라운딩(Radius)을 도면에 맞게 설정.

3
Intermediate Points → Natural, Number: 3

스플라인의 Intermediate Points를 Natural로 설정하고 Number를 3 정도로 낮춰 선을 정리. 불필요한 폴리곤 방지.

4
Extrude로 돌출 + Z축 두께 조정

Extrude 안에 스플라인을 넣고 돌출. Right 뷰에서 Z축 방향으로만 커지므로, Rectangle 스플라인의 Z Position을 조정해 두께를 도면에 맞춤.

5
Extrude의 Cap 끄기 + Make Editable

Extrude의 Cap(앞·뒷면) 옵션을 모두 꺼 테두리 형태만 남김. 준비가 되면 Make Editable(C키)로 편집 가능한 메쉬로 변환.

6
1/4만 남기고 나머지 삭제

아이폰은 좌우·상하 대칭이므로 1/4만 남기고 나머지 면을 전부 삭제. 작업량을 4분의 1로 줄이고 대칭을 활용.

7
Symmetry로 좌우·상하 복사

1/4 테두리를 Symmetry에 넣어 좌우 복사 → 그 Symmetry를 다시 Symmetry에 넣어 상하도 복사. 최종적으로 완전한 테두리 완성.

8
도면 보며 엣지 Extrude로 디테일 추가

도면을 참고해 버튼, 포트 등의 디테일을 Edge Extrude로 추가해 완성.

대칭 모델링의 핵심 대칭이 있는 오브젝트는 항상 최소 단위(1/2 또는 1/4)만 작업하고 Symmetry로 복사합니다. 수정 시에도 원본 1/4만 수정하면 전체에 반영되어 작업 효율이 4배 향상됩니다.
