영상 창작물 저작권 — 폰트가 핵심

영상을 공개·배포할 때 저작권 위반이 가장 많이 발생하는 요소 중 하나가 폰트다. 음악·이미지보다 간과하기 쉽지만 실제로는 매우 엄격하게 적용된다.

폰트 저작권은 왜 빡빡한가?

폰트는 폰트 제작사가 별도로 라이선스를 판매하는 소프트웨어다. 내 컴퓨터에 설치되어 있다고 해서 상업 영상에 무료로 사용할 수 있는 것이 아니다. 영상에 삽입된 순간 배포·공개 라이선스가 별도로 필요하다.

🏰

디즈니

자사 캐릭터·음악·영상에 대해 세계에서 가장 엄격하게 저작권을 관리하는 회사 중 하나. 상업적 사용은 물론 비상업 영상에도 적극적으로 제재.

🎬

미야자키 하야오 · 신카이 마코토

지브리·코믹스웨이브의 작품은 이미지·배경·음악 모두 엄격하게 보호. 팬아트도 상업 영상에 무단 사용 불가.

실무 원칙 영상에 쓰이는 모든 에셋 — 폰트, 음악, 이미지, 영상 클립 — 은 반드시 상업적 사용 허가된 라이선스를 확인한 후 사용한다. "무료" ≠ "상업 무료"임을 항상 기억할 것.
리소스

한글 폰트 다운로드 사이트

국내 한글 폰트를 구할 수 있는 대표 사이트 3곳. 라이선스 범위가 각각 다르므로 사용 전 확인 필수.

무료 위주

눈누 (noonnu.cc)

무료로 사용 가능한 한글 폰트만 모아놓은 전문 사이트. 상업적 사용 여부를 폰트별로 필터링해서 검색할 수 있어 실무에서 가장 많이 활용된다.

  • 모든 폰트가 무료 (유료 폰트는 등록 안 됨)
  • 상업용 가능 여부 명확히 표시
  • 미리보기 텍스트 입력 기능 제공
부분 무료

윤디자인 (yoondesign.com)

국내 대표 폰트 전문 회사. 일부 폰트는 무료 제공하지만 대부분 유료 구독이 필요하다. 고급 브랜딩·인쇄용 폰트가 많다.

  • 전문 타이포그래피 퀄리티
  • 기업 라이선스 별도 구매 필요
부분 무료

산돌구름 (sandollcloud.com)

국내 최대 폰트 라이브러리 중 하나. 구독형 서비스로 운영되며 일부 폰트는 무료. 월정액으로 프리미엄 폰트 전체 사용 가능.

  • 폰트 수량이 매우 많음
  • 구독 시 업무용까지 커버

폰트 포맷

OTF vs TTF — 어떻게 다른가?

결과적으로 같은 폰트지만 파일 구조와 기능에 차이가 있다. 용도에 맞게 골라 사용하자.

TTF True Type Font OTF Open Type Font
개발 Apple + Microsoft Adobe + Microsoft (TTF 확장)
파일 구조 단순 더 확장된 타입
지원 문자 기본 문자 위주 다양한 문자 / 대체 문자 지원
타이포그래피 기능 제한적 합자(Ligature), 스와시 등 풍부함
파일 크기 비교적 작음 비교적 큼
호환성 매우 높음 ✅ TTF보다 낮을 수 있음

TTF 선택할 때

글자가 잘 보이기만 하면 되는 용도 — 웹 본문, 일반 문서, 시스템 폰트. 호환성이 최우선일 때.

웹 · 일반 문서

OTF 선택할 때

고급 타이포그래피 기능이 필요한 용도 — 브랜딩, 인쇄물, 로고, 디자인 작업. 폰트의 미세한 디테일까지 살려야 할 때.

디자인 · 브랜딩 · 타이포그래피
실무에서의 선택 After Effects나 C4D 등 모션그래픽 작업에서는 대부분 OTF를 선호한다. 타이포그래피 디테일이 풍부하고 고화질 렌더 시 더 정교하게 표현되기 때문. 단, 특정 플러그인이나 구형 시스템과의 호환이 필요하면 TTF도 유용하다.
영문 폰트

Dafont & Helvetica

영문 폰트가 필요할 때 찾는 대표 사이트와, 세계에서 가장 유명한 영문 폰트.

Dafont (dafont.com)

다양한 스타일의 영문 폰트를 무료로 다운로드할 수 있는 사이트.

  • 개성 강한 디스플레이 폰트 다수
  • 폰트별로 개인용 / 상업용 라이선스 구분 표시
  • 국내 외 타임스퀘어 같은 공식 광고가 아닌 이상 대부분 사용해도 큰 문제 없음
  • 단, 상업 프로젝트라면 폰트별 라이선스 반드시 확인

Helvetica

1957년 스위스에서 만들어진 영문 폰트. 세계에서 가장 많이 사용되는 산세리프 폰트.

  • Apple, NASA, 지하철 안내판 등 수많은 브랜드가 사용
  • 깔끔하고 중립적이어서 어디에나 잘 어울림
  • Mac에는 기본 내장 / Windows는 별도 구매 필요
  • 무료 대안: Arial (TTF, 유사 형태)
모션그래픽 기초

모션 작업 순서 — 레이아웃 → 주모션 → 부모션

모션그래픽의 기원은 디즈니 애니메이션. 디즈니가 정립한 움직임의 원칙이 현대 모션그래픽의 토대가 됐다.

01
레이아웃 잡기

움직임을 주기 전에 먼저 화면 구성(레이아웃)을 완성한다. 텍스트·이미지·그래픽 요소의 크기·위치·색상이 확정된 후 모션 작업을 시작해야 수정이 줄어든다.

레이아웃이 흔들리면 모션을 다 짜고 나서도 전부 다시 해야 하는 상황이 생긴다.
02
주모션 설정

피사체가 움직일 때 가장 메인이 되는 움직임. 오브젝트가 어디서 어디로 이동하는지, 어떻게 등장하고 퇴장하는지를 먼저 잡는다.

주모션 = 이야기의 뼈대. 이것만 있어도 영상이 읽힌다.
03
부모션 추가

주모션을 더 멋있게 보이기 위해 추가적으로 부여하는 보조 움직임. 스케일 바운스, 회전, 흔들림, 위성 요소의 작은 움직임 등이 해당.

부모션 = 살과 디테일. 없어도 읽히지만, 있으면 훨씬 완성도가 올라간다.
주모션 부모션
역할 가장 메인이 되는 움직임 주모션을 강조하는 보조 움직임
중요도 필수 — 없으면 영상이 성립 안 됨 선택 — 없어도 되지만 있으면 완성도↑
예시 텍스트가 왼쪽에서 날아와 화면 중앙에 안착 안착 직후 살짝 튀어오르는 바운스 + 그림자 생성

오늘 수업 핵심 정리

  • 폰트 저작권 — "무료 설치"와 "상업 영상 사용 가능"은 다르다. 반드시 라이선스 확인.
  • 눈누 = 무료 한글 폰트 허브 / 윤디자인·산돌구름 = 유료 전문 폰트
  • OTF = 고급 타이포그래피·디자인 작업 / TTF = 호환성 우선·일반 사용
  • 모션 작업 순서: 레이아웃 → 주모션 → 부모션 순으로 진행
